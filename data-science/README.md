# MONEDIX - Data Science

Carpeta destinada a los modelos de Ciencia de Datos y Machine Learning.

## Estructura
- `/notebooks`: Notebooks de Jupyter para exploración y entrenamiento.
- `/models`: Modelos serializados (pickle, onnx).
- `/scripts`: Scripts de entrenamiento y evaluación.

## Stack
- XGBoost
- K-Means
- Prophet
- MLflow
